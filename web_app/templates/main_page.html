How to play:

  • Move via arrows or w,a,s,d
  • Access inventory via 'I'
  • Attack via space (the range is 2 tiles forward and ttacks are int he facing direction)
  • Shoot blocks via 'Q' (works the same with melee as bolts go in the direction you face)
  • Shops are the orange blocks and are accessed via walking up to them and pressing 'E'
  • To close menus press 'ESC'
